  ## DESCRIPTION
  ### This social media back-end application allows for proper routing and creation of thoughts and users. A front end application can use the given code to include users and their thoughts within their social media application.

  ## INSTALLATION
  ### To install this application the user must clone the repo given to their local computer. After this they are then able to npm install and npm start to have the application start listening.

  ## USAGE
  ### In order to use this application the user must test the routes within insomnia. Here is a link to a demonstration of the application testing within insomnia: https://drive.google.com/file/d/1SczeysmQiQ2mzljVba0i09_ITfb52-OO/view
